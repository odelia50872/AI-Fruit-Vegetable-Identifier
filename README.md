# FreshScan: AI Fruit & Vegetable Identifier

**FreshScan** is an intelligent system for identifying fruits and vegetables, combining advanced computer vision technologies for automatic detection, counting, and price estimation.

# System Overview
The system provides a fast and accurate solution for item identification at the point of sale or through an interactive user interface:
*   **High Accuracy:** The model achieves an accuracy rate of 95%.
*   **Speed:** Average processing time of less than 3 seconds.
*   **Diversity:** Identification of over 34 different types of fruits and vegetables.

The system performs in-depth image analysis, detects the objects, calculates their estimated weight, and displays a final price summary.

The system is based on the **YOLOv8** architecture and is implemented using Python, Streamlit, and image processing libraries such as OpenCV.
