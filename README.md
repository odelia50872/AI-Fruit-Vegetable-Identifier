# FreshScan: AI Fruit & Vegetable Identifier

**FreshScan** היא מערכת חכמה לזיהוי פירות וירקות, המשלבת טכנולוגיות ראייה ממוחשבת מתקדמות לזיהוי, ספירה והערכת מחיר אוטומטית.

## סקירת המערכת
המערכת מעניקה פתרון מהיר ומדויק לזיהוי פריטים בנקודת המכירה או בממשק משתמש אינטראקטיבי:
*   **דיוק גבוה:** המודל מציג רמת דיוק של 95%.
*   **מהירות:** זמן עיבוד ממוצע של פחות מ-3 שניות.
*   **מגוון:** זיהוי של מעל 34 סוגי פירות וירקות שונים.

![דף הבית של המערכת](צילום_מסך_2026-07-12_232535.jpg)

## יכולות המערכת
המערכת מבצעת ניתוח עומק של התמונה, מזהה את האובייקטים, מחשבת את משקלם המוערך ומציגה סיכום מחיר סופי.

![תהליך זיהוי אבוקדו](צילום_מסך_2026-07-12_232505.jpg)

כמו כן, המערכת כוללת מחירון מובנה המאפשר ניהול ומעקב אחרי מחירי הפריטים הנתמכים.

![מחירון המערכת](צילום_מסך_2026-07-12_225951.jpg)

## טכנולוגיות וארכיטקטורה
המערכת מבוססת על הארכיטקטורה של **YOLOv8** ומיושמת באמצעות Python, Streamlit וספריות עיבוד תמונה כגון OpenCV.
