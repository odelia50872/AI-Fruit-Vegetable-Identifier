# FreshScan: AI Fruit & Vegetable Identifier

**FreshScan** היא מערכת חכמה לזיהוי פירות וירקות, המשלבת טכנולוגיות ראייה ממוחשבת מתקדמות לזיהוי, ספירה והערכת מחיר אוטומטית.

# סקירת המערכת
המערכת מעניקה פתרון מהיר ומדויק לזיהוי פריטים בנקודת המכירה או בממשק משתמש אינטראקטיבי:
*   **דיוק גבוה:** המודל מציג רמת דיוק של 95%.
*   **מהירות:** זמן עיבוד ממוצע של פחות מ-3 שניות.
*   **מגוון:** זיהוי של מעל 34 סוגי פירות וירקות שונים.
